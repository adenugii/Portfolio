## ‚ö° Engineering Highlights

A few technical details implemented in this project:

- **Physics-Based Interactions:** Custom `SpotlightCard` component that tracks mouse coordinates in real-time to create a dynamic lighting effect.
- **Performance Optimization:** Heavy computations (like the dot-grid background) are optimized using event listeners attached to the window object to prevent unnecessary React re-renders.
- **Responsive Design System:** Complex layouts (like the Bento Grid) automatically adapt to mobile viewports with a custom touch-friendly navigation drawer.
- **Scroll Spy Logic:** Custom React Hook (`useActiveSection`) that calculates viewport intersection to update the navigation state dynamically.
